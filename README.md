# AI Movie Recommendation System

A Streamlit web app that lets users search for a movie and get similar movie recommendations using the TMDB API.

## Features

- Search movies
- View poster, rating, overview, and release date
- Get similar movie recommendations
- Uses CountVectorizer and cosine similarity

## Tech Stack

Python, Streamlit, TMDB API, requests, pandas, scikit-learn

## Live App

https://movie-recommendation-system-hh4udwdptbglaedfguylqd.streamlit.app/

## Note

The Colab notebook shows the step-by-step API and recommendation logic.
